# 03. Requirements Traceability Matrix

## 1. Purpose

이 문서는 MSG Foundation 파일럿 시스템의 요구사항 추적성을 관리한다. 이전 버전은 설계 산출물 링크와 ID를 중심으로 구성되어 있었으나, 이 버전은 첨부 PDF의 요구사항 내용을 읽을 수 있도록 요구사항 자체를 서술식으로 정리한다.

Traceability는 다음 흐름으로 관리한다.

```text
PDF source requirement -> 분석 요구사항 -> 설계 책임 -> 검증 기준
```
